# Agensi.io Current State — April 2026

## Key Findings

**Agensi.io is actively operational** — far from the ghost town status in March 2026 research.

- **200+ curated skills** in the marketplace (up from minimal activity before)
- **Learning center active** with guides on monetization, token reduction, SKILL.md templates
- **Weekly winners leaderboard** and skill request/bounty system operational
- **Multi-agent compatibility** — skills work across Claude Code, OpenClaw, Codex CLI, Cursor, and 20+ agents
- **Pricing model**: "Buy once, install instantly, own forever" (no recurring subscriptions mentioned)

**Still opaque**: Team/company info, social media presence, and SimilarWeb traffic rank not discoverable through public search.

## Current State

### Site Status
Live and maintained. Homepage highlights:

**Skill Categories:**
- Frontend & Design
- Testing & QA
- DevOps & Deployment
- Code Review
- Documentation
- Productivity
- Data Engineering
- API Development

**Sample Skills:** code-reviewer, sprint-planner, security-audit, pr-description-writer, api-mock

### Positioning
Agensi positions itself as the **curated, security-reviewed marketplace** vs. raw GitHub directories. Their value prop: pre-screened skills with learning resources + community features (leaderboards, bounties).

### Ecosystem Role
Beneficiary of SKILL.md standardization:
- Anthropic adopted SKILL.md as the open standard
- Microsoft/GitHub Copilot adopted the same format (skills now portable)
- Skills published on Agensi can serve Claude Code, Copilot, Cursor, and 20+ agents

This is **structural validation** — not a niche platform, but part of the emerging multi-agent skill economy.

## Other Marketplaces Found

### Claude-Specific Skill Marketplaces

1. **SkillsMP** ([skillsmp.com](https://skillsmp.com))
   - 66,541+ skills aggregated (Jan 2026)
   - Independent community project (not Anthropic-owned)
   - Displays 800,000+ compatible skills across Claude Code, Codex CLI, ChatGPT
   - Lighter-touch curation vs. Agensi
   - GitHub-sourced discovery

2. **Agent37** ([agent37.com](https://www.agent37.com))
   - **Monetization-focused** — hosted platform for selling skills as products
   - Integrated payments, no Stripe setup required
   - Turn a skill from "local artifact" to "live product with payments" in one day
   - Value: removes friction for skill creators going commercial

3. **SkillHub** ([skillhub.club](https://www.skillhub.club/))
   - Another Claude Code skills marketplace
   - Less prominent in search results (younger or lower traffic)

4. **MCP Market** ([mcpmarket.com](https://mcpmarket.com))
   - Broader MCP tools directory (Model Context Protocol)
   - Skills listed as one category within larger MCP ecosystem

5. **LobeHub** ([lobehub.com/skills](https://lobehub.com/skills))
   - Agent Skills Directory for Claude, Codex, ChatGPT
   - Multi-agent compatible

6. **Anthropic's Official Claude Marketplace** (Limited Preview — March 2026)
   - Enterprise procurement platform launched March 6, 2026
   - **Not** for individual skill creators — for enterprise AI workflows
   - Partners: GitLab, Harvey, Lovable, Replit, Rogo, Snowflake
   - Works against Anthropic commitment spend
   - Limited preview (not open to the public yet)

### GitHub Communities

- **daymade/claude-code-skills** — professional production-ready skills
