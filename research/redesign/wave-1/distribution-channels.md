# ClaudeFlows — Distribution Channels Research
**Wave 1 | April 2026**

This report covers channels the existing brief partially or fully missed. The brief touched Reddit, HN, Instagram, Twitter, LinkedIn, and Upwork. Everything below is additive.

---

## Key Findings

1. **The Claude Code skills ecosystem exploded in late 2025.** GitHub repos with 22K+ stars, "awesome-claude-skills" lists, curated skill directories — there's a living community already hunting for exactly what ClaudeFlows offers. Being present there is table-stakes, not a stretch goal.

2. **Discord is the highest-leverage low-cost channel.** The official Anthropic Discord has 85K+ members. Anthropic just shipped Claude Code Channels (Discord-native). The community actively shares skill recommendations. A genuine presence here has outsized impact for near-zero cost.

3. **Newsletter sponsorships work but are expensive.** TLDR charges $5K–$15K per placement. The Pragmatic Engineer newsletter (the highest-signal outlet covering Claude Code) likely runs $3K–$8K. Budget is the bottleneck, not relevance.

4. **Dev.to is underused and highly relevant.** Claude Code tutorials on Dev.to are going viral. A well-timed "10 Claude Code skills worth installing" article or a tutorial using ClaudeFlows installs costs nothing and targets exactly the right audience.

5. **ProductHunt is a one-shot amplifier, not a growth engine.** 2.7M monthly uniques, AI tools category is competitive but still relevant. Best used as a credibility stamp + email capture spike — not sustainable traffic. Needs preparation (500+ followers, video demo, launch day support).

6. **YouTube is a warm channel, not a cold one.** Reaching out to Claude Code tutorial creators (Matt Wolfe, Skill Leap AI, AI Advantage) for an organic mention or sponsor slot is realistic. These are small-to-mid creators who respond to outreach.

7. **Telegram has active developer communities.** Anthropic officially supports Claude Code Channels via Telegram. There are active developer groups building skills and MCP tools — a native channel for the core user.

8. **Facebook and WhatsApp are low-relevance.** Non-technical AI users don't install CLI tools. These audiences don't convert for ClaudeFlows.

## Detailed Analysis

### 1. Discord

**Key communities:**
- **Official Anthropic Discord** — 85,469 members. The `#claude-code` and `#developer-tools` channels are active. Anthropic launched Claude Code Channels as a first-class Discord feature in March 2026, meaning the community is primed to talk about skills and workflows.
- **AI tinkerers / indie hacker servers** — Dozens of mid-size (1K–20K member) servers exist around AI building: Latent Space, Lenny's community, various no-code/AI builder servers.
- **MCP Market community** — mcpmarket.com lists Claude skills and has associated Discord activity.

**Approach:** Don't drop links. Contribute genuinely — answer skill questions, share one skill that solves a real pain, post updates when new skills ship. Embed a "get it via ClaudeFlows" CTA only after establishing presence.

| Audience size | Relevance | Effort to enter | Expected impact |
|---|---|---|---|
| 85K (Anthropic) + fragmented others | Very high | Low (free, just join) | High |

### 2. Slack Communities

**Key communities:**
- **Indie Hackers Slack** — Multiple channels; there's an #ai-tools and #building-in-public vibe. Not huge (the AI subset Slack had 75 members as of last data point) but high-intent.
- **Lenny's Slack** — Product-focused, some developer crossover. Paid membership, which filters for serious builders.
- **Various "AI for work" Slacks** — Several with 500–5K members targeted at founders using AI tools in business.

**Approach:** These communities are moderation-heavy. Cold promotion gets banned. Strategy is to share genuinely useful content (a skill that solves a real problem for their audience), participate in threads, and let word-of-mouth do the work. Lenny's is worth the membership cost for the exposure.

| Audience size | Relevance | Effort to enter | Expected impact |

| 500–10K across relevant servers | Medium-high | Medium (moderation gatekeeping) | Medium |

### 3. Newsletters

**High-signal outlets covering Claude Code:**

- **Pragmatic Engineer** (Gergely Orosz) — Covers AI tooling for software engineers. Research shows this newsletter explicitly covered Claude Code as "the most-used AI coding tool, overtaking Copilot and Cursor." Audience: senior engineers, engineering managers. ~600K subscribers. Sponsorship: likely $3K–$8K. Editorial mention: extremely hard to earn but possible with a compelling angle (e.g., "skills marketplace for Claude Code").
- **The AI Corner** — Claude-specific coverage. Smaller, more accessible. Reaches Claude power users directly.
- **AI Maker / Prosper (Substack)** — Published comprehensive Claude Q1 2026 guide. Good editorial match.
- **XDA AI Insider** — Weekly AI tool recommendations; has reviewed Claude Code. Accessible editorial contact.
- **TLDR Newsletter** — 1.6M subscribers (flagship) + 1.25M (TLDR AI). Sponsorships: $5K–$15K per slot. ROI depends on conversion rates from a developer audience. Best used for a launch push, not ongoing.
- **Hacker Newsletter** — Curates top HN posts weekly. Getting on HN is the prerequisite — the newsletter is a downstream amplifier.
- **Developer newsletter list on GitHub** — jackbridger/developer-newsletters is a curated list of developer newsletters with sponsorship contacts.

**Approach for editorial (free) mentions:** Pitch a hook, not a product. "We built a SKILL.md marketplace — here's what the top 10 skills tell us about how developers actually use Claude Code" is a story. "ClaudeFlows is a marketplace" is not.

### 7. ProductHunt

**Current state of the market:**
- 2.7M monthly unique visitors, 91 domain rating (Ahrefs)
- AI tools category is active and competitive but not saturated for dev tools
- Cursor launched 5 times in 2025 and won Product of the Year 2024
- v0 by Vercel launched twice, won #1 Product of the Day

**Realistic expectations for a dev tool launch:**
- Top 5 finish: ~200–350 upvotes needed
- Traffic spike: 2,000–10,000 site visits on launch day
- Email captures: 100–500 if gate is in place (ClaudeFlows already has email gate)
- Longevity: Traffic decays to near zero within 72 hours

**What works:**
- Video demo is mandatory — products without video average 167 upvotes vs. 384 with video
- Maker's first comment is the highest-impact single asset
- Need 500+ PH followers pre-launch
- Launch Tuesday–Thursday for max traffic
- DM campaign on launch day can generate 200–300 upvotes

**Strategy:** Use PH as a credibility stamp and initial email list builder — not as a sustained traffic source. Coordinate with Discord / Twitter community to drive launch day votes.

## Actionable Takeaways

1. **Start with GitHub.** Submit to awesome-claude-skills repos today. Tag all skill repos with `claude-code-skills`. This is the channel where the most motivated buyers are already searching.

2. **Join the Anthropic Discord and be human.** Don't announce ClaudeFlows — show up, answer questions about skills, and let word-of-mouth happen. The 85K members include many early adopters who influence others.

3. **Write one Dev.to article per week for a month.** The format that works: "I built [specific outcome] using Claude Code + these 3 skills" or "The 10 Claude Code skills I install on every project." Concrete, opinionated, not promotional.

4. **Telegram is underrated.** Claude Code Channels runs on Telegram. The developer community is there. Join the most active groups, contribute genuine value, and mention ClaudeFlows when it's directly relevant.

5. **Use ProductHunt as a launch event, not a growth channel.** Build a PH following now. Launch with video, a strong maker comment, and coordinate your Discord/Twitter community. Expect a traffic spike and email capture, not sustained referral traffic.

6. **Newsletter sponsorships: test small first.** Find developer newsletters in the $200–$2K range (jackbridger/developer-newsletters is a good starting list) and run a test before committing to TLDR-scale spend.

7. **YouTube outreach is a 60-day play.** Identify 10 mid-size Claude Code creators, watch their recent videos, reference something specific in outreach, offer genuine value (a custom skill demo, co-creation). Budget $500–$3K per integration if going paid.

8. **Podcast guesting builds long-term authority.** Start small (Built This Week, Code and Conquer), develop a 20-minute talk track, use those appearances to pitch Indie Hackers and Lenny's.
