# New Claude Code Skill Sellers — April 2026

Research conducted: 2026-04-16
Scope: Post-March 2026 entrants; platforms, products, prices, evidence of real sales

---

## Key Findings

1. **The market exploded since March 2026.** What was a ghost town (Agensi.io, Claw Mart) is now a crowded, noisy space. At least 10 new marketplaces or directories launched. Most are directories masquerading as marketplaces — free or nearly free, monetizing on sponsorships or API proxy fees.

2. **One platform has verified revenue: Agent37** — $5,819/month (March 2026, verified via Stripe API, tracked on TrustMRR). They are the clearest evidence of a working business in this space. Launched Product Hunt March 14, 2026, hit #1 on launch day with 418 upvotes and 651 followers.

3. **Paid skill pricing is remarkably low.** The only concrete paid skill found is Git Dojo at 5 EUR. Gumroad courses about Claude Code (e.g., "Claude Code for Designers" at $49.99 with 42 reviews) outperform skill-native products.

4. **No marketplace has meaningful paid GMV yet.** Every platform that looked like a transactional marketplace on the surface turned out to be either: (a) a free directory, (b) pre-launch waitlist, or (c) selling platform subscriptions ($9.99/mo for API access), not skill-specific revenues.

5. **The agency "skill access" model is emerging but not yet commoditized.** Marketing consultants are positioning skills as productized deliverables, but there's no verified public case with hard revenue numbers other than Agent37.

6. **LobeHub has the largest collection** (217,527+ skills claimed, 73,800 GitHub stars), but operates as a free open-source platform — no commercial transactions.

## Detailed Analysis

### Agent37 — The Most Concrete Competitor

- **URL:** agent37.com
- **What it is:** Hosted OpenClaw container + skill monetization platform. Builders upload a skill, get a hosted URL, and Agent37 handles payments via Stripe.
- **Pricing (for buyers):** $3.99/month for hosted OpenClaw instance. Revenue share: creators keep 80%, Agent37 takes 20%.
- **Revenue:** $5,819/month verified (Stripe API-confirmed, TrustMRR, ~March 25 2026 measurement). This is platform MRR, not individual skill sales.
- **Product Hunt:** Launched March 14, 2026. #1 on launch day. 418 upvotes, 651 followers.
- **Founders:** Vishnu (@vishnukool), Amanda Silmon (@amanda_silmon)
- **Significance:** This is the only player with public, verified MRR and a real deployment story. Their model — host + pay — is essentially what ClaudeFlows is trying to build.

### SkillHub.club

- **URL:** skillhub.club
- **What it is:** "Universal AI Agent Skills marketplace" — 87,075 skills collected, supports Claude Code, Cursor, Gemini CLI, Codex and 30+ platforms.
- **Monetization:** Pro subscription at $9.99/month (50 daily playground queries). Pay-as-you-go credits and bring-your-own-API-key options. NOT individual skill sales.
- **No transactional marketplace** for skill purchases. Revenue comes from developers paying for API proxy access.
- **Significance:** Large collection, cross-platform angle is smart. But they're selling access to Claude, not skills.

### LobeHub Skills

- **URL:** lobehub.com/skills
- **What it is:** Massive open-source AI platform (73,800 GitHub stars). 217,527+ skills in their directory.
- **Pricing:** Entirely free. Skills are open-source.
- **Significance:** The GitHub of skills — sets a free floor that makes charging for individual skills harder.

### ClaudeSkills.ai

- **URL:** claudeskills.ai
- **Status:** Pre-launch / waitlist. "Coming Soon."
- **Proposed model:** 90% revenue to sellers via Stripe Connect.
- **No founder named.** Zero traction evidence.

### ClaudeSkillsMarket.com (Small Business focus)

- **URL:** claudeskillsmarket.com
- **What it is:** Free directory of 37+ community skills + 14-week certification course + job board for skill commissions.
- **Pricing:** Free browsing. Unclear if certification costs anything.
- **Founded 2025.** No founder named publicly.
- **Significance:** Targets agencies and small businesses, not devs. Same positioning space as ClaudeFlows.

### ClaudeMarketplaces.com

- **URL:** claudemarketplaces.com
- **What it is:** Curated directory/registry — not a transactional marketplace. Free, ad-supported. Sponsorships at $X per 100K monthly developer views.
- **Built by:** @mertduzgun (independent, not affiliated with Anthropic).
- **Significance:** Aggregator/SEO play, not a real market.

### Agensi.io — Updated Status

- **Still operating** as of April 2026. Pivoted from attempted marketplace to comprehensive educational content (how-to guides for installing skills, SKILL.md vs CLAUDE.md etc.) + a security-reviewed skill directory.
- **Security scan on every listing** (8-point automated scan) is their differentiator.
- **Referral/commission model exists** but no public pricing on paid skills.
- **Assessment:** No longer a ghost town in traffic terms, but still not a true paid marketplace.

- **hesreallyhim/awesome-claude-code** — The primary curation list. Slash commands, hooks, skills, orchestrators.
- **rohitg00/awesome-claude-code-toolkit** — #1 trending GitHub repo February 2026. Claims 135 agents, 400K+ skills via "SkillKit."
- **alirezarezvani/claude-skills** — 232+ skills across engineering, marketing, product, C-level. Cross-platform (Claude, Codex, Gemini, Cursor, 8 others).

- **tradermonty/claude-trading-skills** — Trading/investing vertical. Skills for backtesting, screening, market analysis.

All free. GitHub repos function as lead gen for consulting, not direct revenue.

## Actionable Takeaways

**For ClaudeFlows:**

1. **Agent37 is the direct competitor to watch.** ~$5.8K MRR, marketplace + hosting model, 80/20 split. Study their onboarding and pricing. They launched 5 weeks ago and already have verified revenue — they're executing fast.

2. **The pricing ceiling for individual skills is lower than expected.** 5 EUR is the high water mark for a standalone paid SKILL.md. Courses ($49) convert better than skills ($5). This suggests ClaudeFlows needs to bundle or productize — not sell raw skills.

3. **Nobody owns the "paid skills marketplace" yet.** ClaudeSkills.ai is pre-launch with no traction. Agensi.io is a directory. The transactional layer is wide open, but Agent37 is moving fastest.

4. **Cross-platform is a real angle.** SkillHub.club and LobeHub both support Claude + Cursor + Gemini + Codex. SKILL.md format is increasingly cross-platform. A marketplace that handles format compatibility could win.

5. **Verticals are proliferating fast** (trading skills, DevOps skills, YouTube creator skills, marketing skills). A vertical-first marketplace strategy may beat a horizontal one.

6. **GitHub is the distribution channel right now.** The actual discovery happens via awesome-claude-code lists and GitHub Topics. Whatever ClaudeFlows builds needs a GitHub install path.

7. **Security review is a real differentiator.** Agensi.io's 8-point automated security scan is smart. Claude Code skills execute arbitrary code — buyers need trust signals.
