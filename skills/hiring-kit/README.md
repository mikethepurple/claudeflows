# Job Description + Interview Kit

Everything you need to hire well, in 5 minutes

## What it does

Hiring someone means writing a job description, designing interview questions, building a scorecard, and creating a take-home assignment. This skill generates all of it from a role title and seniority level, saving hours of blank-page staring.

## Install

```bash
claudeflows install hiring-kit
```

## Usage

```
claude
> /create-kit
```

Provide the role title, seniority level, team, company name, and work model. Optionally add company values, salary range, or specific requirements.

## What you get

- SEO-optimized job description with inclusive language
- Structured interview scorecard (1-5 scoring, 6-8 dimensions)
- 8-10 behavioral questions with scoring rubrics and follow-up probes
- Time-boxed take-home assignment (2-4 hours) with evaluation rubric
- Offer talking points for closing candidates

## Requirements

- Claude Code

## Example

> /create-kit
> Senior Backend Engineer at Acme Corp, remote, Platform team

Generates a complete JD emphasizing distributed systems experience, a scorecard covering system design and code quality, behavioral questions mapped to collaboration and technical leadership competencies, and a take-home API design exercise with a 3-hour time box.
