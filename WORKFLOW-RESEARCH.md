# AI Workflow Research by Job Function

**Date:** 2026-03-26
**Purpose:** Identify the top 2-3 workflows per job function that people would pay $49-$149 for as Claude Code skills.
**Filter:** High demand + high willingness to pay + Claude can actually do it well (no visual output, no real-time data dependency).

---

## TIER 1 — STRONGEST OPPORTUNITIES (Ship First)

### 1. Competitive Intelligence Report (Marketing)
**Workflow:** Input a company/product name + industry. Claude researches competitors from provided URLs/docs, analyzes positioning, pricing, features, messaging, SWOT, and produces a structured competitive intelligence brief with strategic recommendations.

**Evidence of demand:**
- Klue ($16K/yr), Crayon ($15K/yr), Kompyte ($500-1,500/mo) — proves enterprises pay serious money
- Similarweb competitive package: $125/mo; SEMrush Pro: $119/mo
- Marketing consultants charge $100-$400/hr for competitive analysis; a typical project runs $5K-$50K
- Upwork freelancers charge $50-$150/hr for competitive analysis reports
- 68% of SaaS companies now use competitive pricing analysis tools (Forrester 2026)

**What people currently pay:** $500-$5,000 per one-off report from a freelancer; $125-$1,500/mo for SaaS tools
**Claude feasibility:** HIGH. Text-heavy analysis is Claude's sweet spot. Can analyze provided docs, websites (via fetch), structure frameworks (Porter's Five Forces, SWOT), and produce actionable output.
**Suggested price:** $99

## SKIP LIST — Not Feasible as Claude Code Skills

| Workflow | Why Skip |
|----------|----------|
| AI Design System Generator | Output is visual (Figma files, design tokens). Figma has native AI now. Claude can generate token JSON but not the actual design artifacts. |

## KEY INSIGHT

The sweet spot for Claude Code workflow pricing is the **"consultant gap"**: tasks where freelancers charge $500-$5,000 per deliverable, existing SaaS tools cost $100-$1,500/mo, but a well-prompted Claude workflow can produce 70-80% of the quality at a one-time $49-$149 price point. The buyer is a **solo operator, startup founder, or small team lead** who can't justify the SaaS subscription or freelancer cost but needs professional-quality output.

The workflows that score highest on the demand/willingness-to-pay/feasibility intersection are all **document-generation workflows** — they take structured inputs and produce a long-form, professional document. This is exactly what Claude does best.
